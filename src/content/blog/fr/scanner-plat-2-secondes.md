Fini le temps où compter ses calories prenait 15 minutes par repas. Avec LOWIS, **une photo suffit**.

Dans ce guide, découvrez comment utiliser le scanner IA pour analyser vos repas en 2 secondes — que ce soit un tajine maison, un sandwich du midi, ou votre petit-déjeuner préféré.

## Pourquoi le scan photo change tout

### Avant : La galère du comptage manuel

- Chercher "tajine poulet" dans une base de données
- Trouver 50 résultats différents
- Estimer les portions à l'œil
- Additionner chaque ingrédient
- **Temps : 10-15 minutes**

### Après : Le scan LOWIS

- Prendre une photo
- L'IA analyse
- **Temps : 2 secondes**

C'est aussi simple que ça.

## Comment ça marche techniquement

### L'IA derrière le scan

LOWIS utilise des modèles d'intelligence artificielle avancés pour :

1. **Reconnaître les aliments** dans l'image
2. **Identifier les portions** grâce à des repères visuels
3. **Calculer les nutriments** basés sur notre base de données
4. **Contextualiser** avec votre profil (objectifs, préférences)

### La base de données locale

Contrairement aux apps américaines, LOWIS inclut :

- **+500 recettes marocaines** : Tajine, couscous, pastilla, rfissa, tanjia...
- **Plats français traditionnels** : Quiche, cassoulet, ratatouille...
- **Pâtisseries locales** : Msemen, baghrir, chebakia, cornes de gazelle...
- **Street food** : Maakouda, brochettes, bocadillos...

## Guide étape par étape

### Étape 1 : Préparez votre photo

Pour une meilleure reconnaissance :

✅ **Bon éclairage** — Lumière naturelle idéale
✅ **Vue de dessus** — L'assiette entière visible
✅ **Pas de flou** — Image nette et claire
✅ **Aliments visibles** — Pas de couvercle ou emballage

❌ **À éviter** :
- Photos dans le noir
- Angles extrêmes
- Plusieurs assiettes mélangées
- Plats complètement cachés

### Étape 2 : Prenez la photo

1. Ouvrez l'app LOWIS
2. Tapez sur le bouton "+"
3. Sélectionnez "Scanner un repas"
4. Cadrez votre assiette
5. **Clic !**

L'analyse commence instantanément.

### Étape 3 : Vérifiez et ajustez

L'IA affiche :

- **Les aliments détectés** (ex: "Tajine de poulet aux olives")
- **Les portions estimées** (ex: "1 portion standard")
- **Les calories et macros** (protéines, glucides, lipides)

Si quelque chose n'est pas correct :

1. Tapez sur l'élément à modifier
2. Ajustez la portion (+ ou -)
3. Ou remplacez par un autre aliment
4. Validez

**L'IA apprend de vos corrections** et sera plus précise la prochaine fois.

### Étape 4 : Enregistrez

Un tap sur "Confirmer" et c'est fait !

Le repas est ajouté à votre journal avec tous les détails nutritionnels.

## Astuces pour une meilleure reconnaissance

### Pour les plats composés

Si votre assiette contient plusieurs éléments :

- Étalez légèrement les aliments pour qu'ils soient visibles
- Prenez la photo AVANT de mélanger
- Vous pouvez prendre plusieurs photos si nécessaire

### Pour les plats "cachés"

Tajine avec son couvercle ? Sandwich fermé ?

- Prenez une photo du plat ouvert/déplié
- Ou décrivez-le oralement (LOWIS comprend aussi la voix)

### Pour les recettes maison

Votre grand-mère fait le meilleur couscous du monde ?

- Scannez-le une fois
- Sauvegardez-le comme "favoris"
- Réutilisez-le en un tap les fois suivantes

## Au-delà des calories

Le scan ne s'arrête pas aux calories. LOWIS analyse aussi :

- **Macronutriments** : Protéines, glucides, lipides
- **Fibres** : Important pour la satiété
- **Sucres** : Ajoutés vs naturels
- **Sodium** : Pour ceux qui surveillent le sel

Ces informations alimentent les recommandations personnalisées de votre coach IA.

Encore mieux ! Vos plats récents sont sauvegardés. Un tap et c'est enregistré, sans même scanner.
